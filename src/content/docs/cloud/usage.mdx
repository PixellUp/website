# Usage

Umami allows the user to track a variety of metrics for the month. Currently you can track the metrics below.

- Total events and percentage related to your hobby or pricing tier.
- Total website count
- Top websites by events collected

## Track usage

Navigate to **Billing** to view your usage details.
